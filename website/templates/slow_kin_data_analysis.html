Welcome to the application analyzing slow fluorescence transients! These fluorescence transients, so-called Kautsky kinetics, or NPQ protocols, provide a sensitive method for evaluating the efficiency of electron (e-) transport within the photosynthetic electron transport chain.

How to use this tool:

  1. Select your fluorometer
  2. Upload and analyze slow kinetic files (up to 50 files)
  3. Review the results
  4. Download the results in a summary .xlsx file

Accepted file formats include .CSV and .txt files, as obtained by MULTI-COLOR-PAM / DUAL-PAM or AquaPen / FluorPen fluorometers, respecitvely:

MULTI-COLOR-PAM DUAL-PAM AquaPen FluorPen







Please note that MULTI-COLOR PAM files typically contain large data and may take longer time to process. For faster processing, it is recommended to reduce size of the MULTI-COLOR PAM files. This will also allow to export reduced data to the summary .xlsx file for further processing. Reduced files will contain about 10000 data points.
If you wish to work with raw data, uncheck the box below. For AquaPen / PlantPen, data reduction is not necessary.
Please note that it is only possible to work with one file type at a time, although up to 50 files can be processed within a single analysis. The AquaPen/FluorPen files have identical length, due to defined NPQ protocols by the manufacturer. For the MULTI-COLOR PAM files, it is recommended to process only files measured by identical protocols, as defined by the user/experimenter.

For the MULTI-COLOR PAM files with parameters, following timing is considered for F0 and Fm

3. To estimate state transition rates from MULTI-COLOR PAM measurements, select time points defining State 1 and State 2 intervals

The Fm' data can be interpolated by exponential function, in order to estimate rate of state transition:

1. Exponential decay (typically State 1 -> 2 transition): $$ \begin{multline} \shoveleft F_{m}'\ (t) = F_{m}{'}_{t_0}\ *\ e^{-λ\ *\ τ} \end{multline} $$
2. Exponential growth (typically State 2 -> 1 transition): $$ \begin{multline} \shoveleft F_{m}'\ (t) = F_{m}{'}_{t_0}\ *\ e^{λ\ *\ τ}, \end{multline} $$ where Fm't0 and Fm' are maximum fluorescence yields at the onset and throughout a given actinic light period, t is time and λ is rate of exponential decay or growth (unit: t-1).

Protocols in AquaPen or PlantPen are pre-defined, and timing of dark and actinic light periods is fixed. On the other hand, protocols in Multi-Color PAM or Dual PAM are user-defined with variable timing. Below, timing of individual periods (inducing State 1 or State 2) within the slow kinetic protocol from Multi-Color PAM or Dual PAM can be specified, in order to evaluate state transitions rate.
