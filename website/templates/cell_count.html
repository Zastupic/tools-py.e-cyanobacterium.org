Cell counting


Welcome to the cell counting app!
To count cells in your microscopy image, please follow the instructions below:

  1. Select threshold filter
  2. Enter pixel size
  3. Enter depth of the microscopy chamber
  4. Enter minimal cell diameter
  5. Select and upload image from your computer
  6. Count the cells
  7. Read the result and download the counted image

  8. Optional: If some cells have been missed, a correction can be made by entering the number of missed cells into the respective form and repeating the count. (Also a negative number can be entered)
