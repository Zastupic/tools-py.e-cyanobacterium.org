Welcome to CyanoTools, an interactive and comprehensive web platform designed to support researchers and students working in cyanobacteria, algae, plant biology, and microbial research. Developed by the Department of Adaptive Biotechnologies (DoAB) at the Global Change Research Institute of the Czech Academy of Sciences, CyanoTools provides advanced tools for biological data analysis.

From chlorophyll fluorescence and microscopy to bioreactor control, CyanoTools supports experimental workflows using browser-based tools—no installation needed.

CyanoTools is regularly updated with new features and improvements. For the latest news, visit our development log.

Why use CyanoTools?

🎯

Research-Focused

Tools are tailored for cyanobacterial and algal biology, reflecting widely used methods of Chl a fluorescence, microscopy or spectrofluorometry.

🚀

Fast Processing

Optimized algorithms deliver fast, real-time analysis of complex data — no delays, even with large input files.

📚

Educational Value

The tools include explanations, equations, and definitions of key parameters — making them ideal for educational purpose.

🔒

Secure & Private

Every input file is processed in-memory and never stored. All operations are session-based to ensure maximum confidentiality.

📥

Batch processing

The tools support batch processing of multiple files with results downloadable in a summary .xlsx file.

📱

Cross-Platform

Responsive design adapts to phones, tablets, and desktops. The analysis can be run from lab benches, classrooms, or field sites.

🔧

No Installation

Everything runs in the browser. No software installation or updates needed — just open the site and start analyzing.

🌐

Open and Accessible

Freely available to all users worldwide. Fully transparent tool logic allows reproducibility and validation of results.
