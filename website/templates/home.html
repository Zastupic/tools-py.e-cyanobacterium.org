Open-access tools for photosynthesis research


CyanoTools is an open-access, browser-based platform providing integrated analytical tools for researchers working with photosynthetic organisms — microalgae, cyanobacteria, and higher plants. It covers PAM fluorometry, low-temperature spectroscopy, membrane inlet mass spectrometry, and microscopy-based cell analysis — pre-processing raw instrument files for publication-ready results. All tools run entirely in the browser with no installation, account registration, or server-side data retention.

FAQ

What types of data can I analyze with CyanoTools?
CyanoTools supports data analysis from a the following instruments and file formats:
Chlorophyll fluorescence:
- MULTI-COLOR-PAM, DUAL-PAM: .csv files
- FluorPen, AquaPen, FL 6000: .txt files
Spectrofluorometry:
- Jasco spectrofluorometers: .csv files
MIMS:
- Hiden HPR-40: .csv files
- MS GAS: .asc files
Photobioreactors:
- FMT-150: .xlsx files
Microscopy/image analysis:
- .png, .jpg, .jpeg, .tif, .bmp, .gif files
What types of analysis can I perform?
CyanoTools supports the following types of data analysis:
Chlorophyll fluorescence:
- For details on Rapid light curves, OJIP transients, Slow (Kautsky) kinetics, please see the individual tools with detailed explanation.
Spectrofluorometry:
- For details on Fluorescence Spectra & EEM Analyzer (77 K and room temperature excitation-emission spectra), please see the tool with detailed explanation.
Membrane inlet mass spectrometry (MIMS):
- Visualize dynamics of dissolved or gaseous-phase analytes (e.g., O2, CO2, H2, CH4 etc.)
- Normalize signals to compensate for MIMS system drift, baseline shift, or external disturbances.
Photobioreactors:
- Analyze specific growth rate during batch growth experiments.
- Analyze specific growth rate in tubidostat experiments. Apply statistical methods to identify fully acclimated (i.e. stable) growth rate under given cultivation conditions.
- Analyze dynamics in the measured data during periodic experiments (such as diurnal)
- Process exported excel files for easier subsequent manipulation (using Tidy Data R package.)
Microscopy/image analysis:
- Count cells or fluorescence particles on a microscopy image.
- Identify fluorescence profiles across cells, based on pigment autofluorescence or fluorescence probes.
- Identify size of cells or object of interest on an image.
- Work with both regular and irregular cellular, or object shapes.
Complex platforms:
- For details, please visit the Optimalga or E-cyanobacterium platforms.
Can I upload multiple files or datasets?
Chlorophyll fluorescence and Spectrofluorometry support batch uploads and can analyze up to 100 datasets simultaneously.
MIMS, Photobioreactors and Image Analysis tools are designed for single-file analysis.
Can I use the tools without registration?
Yes. All tools are fully open-access and can be used without registration or login.
Will my data be saved on the server?
Most tools operate entirely within your browser, ensuring that no data is uploaded to the server. However, some tools require file uploads to our servers for processing; in such cases, the files are stored temporarily and automatically deleted shortly after your session ends. We do not retain or access uploaded data beyond what is necessary for immediate analysis.
Is the source code available?
Yes. The source code is freely accessible on GitHub. Contributions are welcome!
Can I request a new feature or tool?
Yes! Suggestions for new tools, analysis types, or device support are welcome. Feel free to email us.
Where can I find tutorials or help?
Each tool comes with detailed usage instructions to guide you through the process.
How do I cite CyanoTools in my publication?
To refer to CyanoTools, please use the following citation style:
"CyanoTools – a web-based toolkit for cyanobacteria, algae, and plants research. Available at: https://www.cyano.tools"
