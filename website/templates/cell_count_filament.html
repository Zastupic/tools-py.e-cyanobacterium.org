Cell counting - filamentous strains


Welcome to the app for counting cells of filamentous strains!
Please follow the instructions below:

  1. Select threshold filter
  2. Enter pixel size
  3. Enter depth of the microscopy chamber
  4. Enter minimal cell diameter
  5. Enter additional parameters as necessary for splitting filaments into individual cells
  6. Select image
  7. Identify and count individual cells within filaments
  8. Read the result and download the counted image

  9. Optional: If some cells have been missed, a correction can be made by entering the number of missed cells into the respective form and repeating the count. (Also a negative number can be entered)
